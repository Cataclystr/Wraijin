HTML Notes

Header Tag

Used to create a Heading like in a book.

Paragraph Tag

Used to create paragraphs.

Italicization and Bold

Use the em tag to Italicize and emphazise the data inside of it, and the strong tag to bring your attention to a concept.

Unordered Lists

Used to create lists with bullet points.

Ordered Lists

Used to create Lists with numbers. The type can also be changed in ordered to make it start on different numbers or with Roman Numerals.

Anchor Tag aka Links

Allows you to create Links to outside websites, or to pages in your own website! Remember, you can nest different types of tags.

Image Tag

Used to put Images on your website!

Tables

Used to create tables. Can be used to format, and present data.
